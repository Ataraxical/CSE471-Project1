Score Format

The score is formatted as an XML document with 3 types of nodes: Scores, Instruments, and Notes.

The score has attributes indicating the beats per minute (bpm) of the score and the number of beats per measure (time signature numerator). The denominator is assumed to be 4 (i.e. a "beat" is a quarter note).

Instruments have attributes indicating the instrument type and if effects are enabled (t = enabled, f = disabled) for that instrument (i.e. gate="t"). Notes have many attributes:

  • measure: the measure the note is to be played in.
  • beat: the beat within the measure that the note is to be played in. Can contain fractional numbers.
  • duration: the duration (in beats) that the note is to be played for.
  • note: the note value to indicate the frequency of the note. Uses sharps, not flats. Formatted as value + octave + sharp (i.e. E3 or E3#).
  • velocity: indicates the intonation with which the note is to be played (l for loud, s for soft).
  • pedal: indicates if the piano's sustain pedal was depressed for this note.
Note: Instruments are children of scores and notes are children of instruments.
